# Simulador de Consumo Marketing Cloud — BancoEstado (versión por mensajes)

Simulador interactivo para proyectar consumo de WhatsApp de BancoEstado, expresado en **mensajes** (no en paquetes).

Cada caso de uso permite seleccionar:
- **Tipo de mensaje** — Promocional (multiplicador 109,76), Transaccional (24,7) o Servicio (0).
- **Cantidad de mensajes** — el número que BancoEstado planea enviar.

El simulador convierte automáticamente a **paquetes de 1.000 créditos** (que es el SKU que se compra) y proyecta el consumo contra el contrato.

## Uso

Abrir `index.html` en cualquier navegador moderno. Sin build, sin servidor — solo HTML + Chart.js por CDN.
